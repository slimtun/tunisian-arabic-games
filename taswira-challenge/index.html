💡 How to Play

Tunisian Arabic (Derja) uses special phonetics. To write it using standard keyboards, locals invented Arabizi—using numbers for sounds that don't exist in Latin letters.

7 Representing ح (e.g. 7outa, toffe7a)
3 Representing ع (e.g. 3asfoura)
q Representing ق (e.g. qattous)
